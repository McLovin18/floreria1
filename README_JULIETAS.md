# Julietas Flowers 🌷

Florería online personalizada en Samborondón, Guayaquil. **"Creamos emociones con cada flor"**

Plataforma de venta de flores, arreglos personalizados, rosas y detalles florales para todos los eventos.

## 🌹 Acerca de Julietas Flowers

**Julietas Flowers** es una florería especializada en:

- **🌹 Rosas Premium** - Rosas rojas, blancas, rosadas y mixtas de alta calidad
- **💐 Arreglos Personalizados** - Diseños únicos para cada ocasión
- **🎁 Flores para Eventos** - Bodas, cumpleaños, aniversarios, condolencias, eventos corporativos
- **🌷 Flores Variadas** - Tulipanes, girasoles, orquídeas y más
- **💌 Detalles Especiales** - Flores + peluche, chocolate, globos, velas aromáticas
- **🚚 Envío a Domicilio** - Entregas rápidas en Guayaquil

**📍 Ubicación:** Samborondón, Guayaquil, Ecuador  
**💘 Eslogan:** "Creamos emociones con cada flor"

## 🚀 Comenzar

### Instalación

```bash
# Instalar dependencias
npm install

# Ejecutar servidor de desarrollo
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.

### Variables de Entorno

Crea un archivo `.env.local`:

```env
NEXT_PUBLIC_FIREBASE_API_KEY=tu-api-key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=tu-auth-domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=tu-project-id
# ... más variables
```
